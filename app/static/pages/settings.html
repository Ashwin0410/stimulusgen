About

Chills Stimulus Generator is a tool for creating and testing audio stimuli designed to induce chills (frisson).

It combines AI-generated text, professional text-to-speech, background music, and advanced audio processing to systematically explore what makes audio emotionally powerful.

Version: 1.0.0

Configuration Notes

API keys are configured via environment variables in the .env file: 

# ElevenLabs
ELEVENLABS_API_KEY=your_key_here

# LLM (at least one required)
ANTHROPIC_API_KEY=your_key_here
OPENAI_API_KEY=your_key_here

# FFmpeg (optional, auto-detected)
FFMPEG_BIN=/path/to/ffmpeg

Restart the server after changing environment variables.
